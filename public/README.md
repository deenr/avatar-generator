# Avatar Generator Icons

This directory should contain the following icon files for the web app:

## Required Files

- `favicon.ico` - 32x32 or 16x16 favicon
- `apple-touch-icon.png` - 180x180 Apple touch icon
- `android-chrome-192x192.png` - 192x192 Android icon
- `android-chrome-512x512.png` - 512x512 Android icon

## Generate Icons

You can generate these icons from a single source image using tools like:
- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)

## Icon Design

Consider using:
- A simple "A" letter mark
- Avatar silhouette
- Colorful geometric shape matching the app theme (green #10b981)
